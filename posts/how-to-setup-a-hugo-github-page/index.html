Getting started with GitHub Pages

  • Creating a GitHub Pages site

Install Hugo

  • Quick Start

Clone Repository

We want to push the generated files in public to GitHub. Thus we clone the repository in there:

git clone https://github.com/<user>/<user>.github.io.git public

Generate new static site

With the following command a new version is generated:

hugo

Publish content to GitHub

Now the changes can be added and pushed to GitHub:

git -C public add --all
git -C public commit -m "<write a meaningful commit message>"
git -C public push
